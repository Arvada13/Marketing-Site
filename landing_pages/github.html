1 Create a site

Create a site in CloudCannon and connect GitHub as a storage provider. Select the repository with your Jekyll website.

2 Synchronize

CloudCannon fetches the files from your repository and keeps them in sync.

3 Set editable elements

Add class="editable" to elements in the HTML to control which elements can be updated inline.

Non-developers make updates directly on the page, what they see is what they get.
