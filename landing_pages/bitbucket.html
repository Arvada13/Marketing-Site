How it Works


1 Enable the add-on

Login to Bitbucket and go to Find new add-ons. Click Install next to CloudCannon.

2 Create a site

Click the CloudCannon button on your Bitbucket Jekyll Repository and create a site. CloudCannon will synchronize your files.

3 Set editable elements

Add class="editable" to elements in the HTML to control which elements can be updated inline.

Your non-technical team make updates directly on the page, what they see is what they get.

All changes are pushed back to your Bitbucket Repository
