1 Enable the add-on

Log in to Bitbucket and go to Find new add-ons. Click Install next to CloudCannon.

2 Create a site

Click the CloudCannon button on your Bitbucket Jekyll repository and create a site. CloudCannon then synchronises your files.

3 Set editable elements

Add class="editable" to elements in the HTML to control which elements can be updated inline.

Non-developers make updates directly on the page, what they see is what they get.

All changes are pushed back to your Bitbucket repository with Git.
