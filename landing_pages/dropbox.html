1 Create a site

Create a site in CloudCannon and connect Dropbox as a storage provider. This creates a folder for this site in Dropbox.

2 Synchronise

Add your Jekyll source files to the folder in Dropbox. CloudCannon fetches your files and keeps them in sync.

3 Set editable elements

Add class="editable" to elements in the HTML to control which elements can be updated inline.

Non-developers make updates directly on the page, what they see is what they get.
